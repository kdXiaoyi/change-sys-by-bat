# Changing SYS but by Bat
#### 介绍
`BATCH(*.Bat)`是一种存在于MSDOS与Windows的脚本，可以快速执行指定的命令，而这个工程让它更有意义(尽管它已被`Powershell(*.ps1)`取代)。你可以通过这个工具快速部署并完成系统更改。
#### 技术规范
Windows x86/x64 带有cmd.exe组件的即可运行。部分功能仅Windows7及更低版本支持(Win10部分版本会弹出缺少组件的问题)。
